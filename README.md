# agent_sandbox

## Install

```
uv venv
source .venv/bin/activate
uv pip install -r requirements.txt
```

## Env setup

Need the follow environment variables 

```
OPENAI_API_KEY=<your_openai_api_key>
LOGFIRE_API_KEY=<your_logfire_api_key>
BASALT_API_KEY=<your_basalt_api_key>
```

## Run

```
python app.py
```
